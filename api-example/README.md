# نمونه API برای اپ VPN

این پوشه فقط یک **نمونه** برای سرو کردن لیست سرورها به اپ است. روی سرور خودتان اجرا کنید و **هیچ پسورد یا اطلاعات حساسی در این فایل‌ها قرار ندهید.**

## نحوه استفاده روی سرور

1. این پوشه را روی سرور کپی کنید (مثلاً با SCP/SFTP یا Git).
2. فایل `servers.json` را ویرایش کنید و سرورهای واقعی (با لینک VLESS/VMess و غیره) را قرار دهید.
3. نصب Node.js در صورت نیاز: `apt install nodejs` (یا از nvm).
4. اجرا: `node server.js` (یا با pm2 برای اجرای دائم).
5. اپ با آدرس پایه `http://IP-SERVER:3000/` صحبت می‌کند؛ در محیط واقعی حتماً **HTTPS** با nginx و گواهی SSL جلوی این سرویس قرار دهید و در اپ `VPN_API_BASE_URL` را به همان آدرس HTTPS تنظیم کنید.

## آدرس در اپ

- اگر این نمونه روی پورت 3000 و دامنه شما `api.example.com` باشد، در GitHub Secret مقدار دهید:
  - `VPN_API_BASE_URL` = `https://api.example.com/vpn/`
- اپ درخواست `GET https://api.example.com/vpn/servers` می‌زند و انتظار آرایه JSON با فیلدهای `id`, `name`, `flag`, `config` را دارد.
